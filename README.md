I created my first dashboard on PowerBI & Tableau using the same dataset which I obtained from Kaggle with the help of BeTechified to analyze the superstore dataset.

The goal was to: 
🖇️Identify which category or sub-category is underperforming. 
🖇️ Visualize the trend of sales and profit overtime.

Key insight obtained:
🖇️Copiers has the highest sum profit by category. 
🖇️The western region has the highest sum of sales by region. 
🖇️ According to months, December has the highest sum of sales and profit.

Tools I made use of while visualizing my dashboard were: Area chart, Pie chart, Slicers, Map, Bar chart, Scattered plot and Card chart.
